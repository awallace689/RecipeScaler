This python3 program takes user-submitted screenshots of recipe ingredient lists and returns the text in
a format suitable for future use in a desktop or web application. It utilizes a combination of Google's
'Tesseract ORC', python3 libraries 'pytesseract' and 're', and python3's built-in word processing.
